# omnilingual-asr Notes

## Installation

- 系統依賴：[libsndfile](https://github.com/facebookresearch/fairseq2?tab=readme-ov-file#system-dependencies)(Mac: brew install libsndfile; Windows may need an additional setup).

```bash
# using pip
pip install omnilingual-asr

# using uv
uv add omnilingual-asr
```

> Note: If any required packages are missing, please run `pip install -r requirements.txt`.

## Inference

```python
python3 main.py
```
